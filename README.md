### 项目介绍
基于微信小程序的校园活动广场，是本人本科的毕业设计，目的是想为了丰富大学生的在生活，和提供一个校园活动的消息管理平台。但由于小程序审核限制，个人版暂时无法上线。
### 2.项目组成
wepy小程序 + express服务 + mongodb数据库
